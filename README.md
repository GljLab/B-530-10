# 智慧园区物业管理系统

## 项目简介

基于 **Spring Boot + MyBatis-Flex + Vue + JWT + MySQL** 的智慧园区/物业管理系统，集成园区信息管理、楼宇管理、楼层管理、房产资源管理、业主档案管理、统计看板等功能，并配套完善的用户角色权限体系，为后续租户管理、物业收费等功能奠定基础。

> [!NOTE]
> **技术说明**: 采用 **MyBatis-Flex** 作为 ORM 框架。MyBatis-Flex 是一个优雅的 MyBatis 增强框架，提供了更简洁的 API 和更好的性能。

## 技术栈

### 后端
- **框架**: Spring Boot 2.7.x
- **ORM**: MyBatis-Flex 1.7.x（原需求 mybatis-flux 为笔误）
- **数据库**: MySQL 8.0
- **安全**: Spring Security + JWT
- **连接池**: HikariCP

### 前端
- **框架**: Vue 3
- **构建工具**: Vite
- **UI 组件**: Element Plus
- **路由**: Vue Router

## 快速开始

> [!TIP]
> 使用 Docker Compose 一键启动，无需本地安装 Java、Node.js 等环境

### 环境变量配置

| 变量名 | 说明 | 默认值 |
|--------|------|--------|
| `JWT_SECRET` | JWT 签名密钥（**必须设置**，至少32字符） | 无（占位值，启动时校验） |
| `CORS_ALLOWED_ORIGINS` | 允许的跨域来源（逗号分隔） | `http://localhost:3000` |

> [!IMPORTANT]
> 生产部署前 **必须** 通过环境变量设置 `JWT_SECRET`，否则后端将拒绝启动。
> 可在项目根目录创建 `.env` 文件供 Docker Compose 读取：
> ```env
> JWT_SECRET=YourSuperSecretKeyAtLeast32Characters!!
> CORS_ALLOWED_ORIGINS=https://your-domain.com
> ```

### Docker 部署（推荐）

```bash
# 一键启动
docker compose up -d --build

# 查看日志
docker compose logs -f
```

## 功能模块

### 系统管理
- ✅ **用户管理** - 用户增删改查、状态切换、密码重置
- ✅ **角色管理** - 角色配置、权限分配
- ✅ **菜单权限** - 菜单树维护、权限标识配置
- ✅ **授权管理** - 基于 RBAC 的权限控制
- ✅ **数据权限** - 基于角色的数据访问控制
- ✅ **JWT 认证** - 无状态 Token 认证机制

### 园区管理
- ✅ **园区概览** - 园区信息总览、设施管理、图片展示
- ✅ **楼宇管理** - 楼宇信息维护、楼宇图片、楼宇类型
- ✅ **楼层管理** - 楼层信息维护、楼层用途、房产数量统计
- ✅ **统计看板** - 园区楼宇楼层统计数据

### 房产资源管理
- ✅ **房产信息管理** - 房产完整信息维护，包含基础信息、属性、产权、状态等
  - 房产编号（全园区唯一）
  - 所属楼宇、楼层（联动选择）
  - 房产类型：住宅/商业/其他，支持多种子类型
  - 面积信息：建筑面积、套内面积、公摊面积（自动计算）
  - 户型、朝向、装修状况
  - 产权信息：产权性质、年限、证号、交付日期
  - 房产状态：待售、已售未交付、业主自住、业主出租、空置、装修中、查封
  - 特殊标识：学区房、人才房、拆迁房
- ✅ **房产批量创建** - 高效批量录入房产
  - 选择楼宇和楼层
  - 设置房号规则（前缀、起始号、结束号）
  - 设置默认属性
  - 预览列表确认
  - 重复编号自动跳过并记录
  - 显示创建结果（成功/失败数量）
- ✅ **房产列表与搜索** - 灵活的查询筛选
  - 房产编号模糊搜索
  - 多条件筛选：楼宇、楼层、类型、状态、户型、面积区间、朝向
  - 表格视图展示
  - 状态颜色标识（待售橙、自住绿、出租蓝、空置灰、装修黄、查封红）
  - 支持排序、分页
- ✅ **房产详情与编辑** - 完整信息展示与维护
  - 详情页：基础信息、产权信息、使用情况、备注信息卡片
  - 状态变更历史时间轴
  - 编辑功能：部分字段不可修改（编号、楼宇、楼层、类型）
  - 状态修改：记录变更日志（时间、操作人、原状态、新状态、原因）
- ✅ **房产导入导出** - 批量数据处理
  - 下载Excel导入模板（含示例数据）
  - Excel导入：数据校验、必填项检查、编号重复检测
  - 导入结果展示：成功/失败数量、失败原因列表
  - 失败数据下载，修正后重新导入
  - 导出当前筛选结果为CSV
- ✅ **房产统计分析** - 多维度统计报表
  - 总体统计：房产总数、各类型数量、总面积
  - 状态分布：各状态数量及占比、入住率、空置率、出租率
  - 户型分布、面积分布、朝向分布
  - 楼宇房产分布及入住率对比

### 房产管理操作流程

**1. 查看房产列表**
- 进入"房产管理" → "房产管理"菜单
- 可通过房产编号搜索，支持模糊匹配
- 可按楼宇、楼层、类型、状态等多条件筛选
- 支持按房产编号、面积、楼层排序

**2. 新增房产**
- 点击"新增房产"按钮
- 填写基础信息：房产编号（必填，唯一）、楼宇、楼层、房产类型
- 填写房产属性：面积、户型、朝向、装修状况
- 填写产权信息：产权性质、年限、证号、交付日期
- 填写特殊标识和备注

**3. 批量创建房产**
- 点击"批量创建"按钮
- 步骤1：选择楼宇和楼层
- 步骤2：选择房产类型
- 步骤3：设置房号规则（前缀、起始号、结束号）
- 步骤4：设置默认属性（面积、户型等）
- 步骤5：预览确认，点击"确认创建"
- 系统自动跳过重复编号，并显示创建结果

**4. 导入房产数据**
- 点击"导入房产"按钮
- 先点击"下载导入模板"获取Excel模板
- 按照模板格式填写数据
- 上传Excel文件，点击"开始导入"
- 查看导入结果，失败数据可下载修正后重新导入

**5. 查看房产详情**
- 点击房产编号链接进入详情页
- 查看基础信息、产权信息、使用情况、备注
- 查看状态变更历史时间轴

**6. 修改房产状态**
- 列表页点击"改状态"按钮，或详情页点击"修改状态"
- 选择新状态，填写变更原因（可选）
- 确认后状态变更记录保存到日志

**7. 查看统计分析**
- 进入"房产管理" → "房产统计"菜单
- 查看总体统计数据
- 查看各维度分布图和数据

### 业主管理操作流程

**1. 查看业主列表**
- 进入"业主管理" → "业主管理"菜单
- 快速搜索：姓名、手机号、身份证号模糊搜索
- 高级筛选：业主类型、入住状态、认证状态、业主标签、房产数量、入住日期范围
- 认证状态颜色标识：未认证（灰）、已认证（绿）、认证失败（红）
- 支持按编号、姓名、入住日期、房产数量排序
- 支持分页（20/50/100条）

**2. 新增个人业主**
- 点击"新增业主"按钮
- 步骤1：选择业主类型（个人/企业）
- 步骤2：填写个人信息：姓名、性别、身份证号、出生日期、民族、婚姻状况
- 步骤3：填写联系方式：手机号、备用手机、邮箱、微信号
- 步骤4：填写居住信息：户籍地址、现居住地址、工作单位、职业
- 步骤5：填写家庭成员：家庭人口数、紧急联系人
- 步骤6：选择业主标签（多选）、填写备注
- 步骤7：关联房产（可选）
- 确认提交，系统自动生成业主编号

**3. 新增企业业主**
- 点击"新增业主"按钮，选择"企业业主"
- 除填写个人业主信息外，额外填写：
  - 企业信息：企业名称、统一社会信用代码、企业类型
  - 法人信息：法定代表人姓名、身份证号
  - 联系人：联系人姓名、职务、手机
  - 企业地址、注册资本
  - 上传营业执照扫描件

**4. 关联房产**
- 业主列表点击"详情"进入详情页
- 点击"关联房产"按钮
- 选择房产（下拉或搜索）
- 设定产权关系：完全产权、共有产权（设置产权比例）、使用权
- 填写产权证号、取得方式、取得日期
- 填写购买价格（选填）、入住日期
- 选择是否自住
- 确认提交

**5. 解除房产关联**
- 业主详情页 → 房产信息卡 → 操作列点击"解除关联"
- 填写解除原因
- 确认后关联状态变为"已转让"，历史记录保留

**6. 业主认证审核**
- 业主列表勾选多个业主，点击"批量认证"，或详情页点击"认证审核"
- 查看业主基本信息和关联房产
- 填写审核意见
- 选择审核结果：
  - 通过：认证状态变为"已认证"
  - 拒绝：认证状态变为"认证失败"，需填写拒绝原因
- 审核记录永久保存，详情页底部时间轴展示

**7. 编辑业主信息**
- 列表页点击"编辑"或详情页点击"编辑"
- 可修改：个人信息、联系方式、居住信息、家庭成员、紧急联系人、业主标签、备注
- 不可修改：业主编号、业主类型
- 企业业主可修改企业信息和营业执照

**8. 导入业主数据**
- 点击"批量导入"按钮
- 点击"下载模板"获取Excel模板
- 按模板格式填写数据（必填：姓名、手机号、身份证号）
- 上传Excel文件（支持拖拽）
- 系统校验：必填项、身份证号格式（18位）、手机号格式（11位）、唯一性
- 查看数据预览（前10条）和校验结果
- 确认导入，查看成功/失败统计
- 失败数据可下载查看原因，修正后重新导入

**9. 导出业主数据**
- 筛选需要导出的业主
- 点击"导出数据"按钮
- 生成Excel文件，包含基本信息和房产统计

**10. 查看业主统计**
- 进入"业主管理" → "业主统计"菜单
- 查看顶部统计卡：业主总数、已认证数、入住率、个人/企业业主数
- 查看图表：
  - 业主类型分布饼图
  - 入住状态分布饼图
  - 认证状态分布饼图
  - 房产持有分布柱状图
  - 入住趋势折线图
  - 标签统计柱状图
